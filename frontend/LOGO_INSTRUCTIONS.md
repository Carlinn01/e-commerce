# Instruções para o Logo

## Localização do Arquivo

O arquivo `LOGO.png` deve ser colocado na pasta `frontend/` junto com os outros arquivos HTML e CSS.

## Estrutura Esperada

```
frontend/
├── LOGO.png          ← Coloque o arquivo aqui
├── index.html
├── styles.css
├── styles-modern.css
└── ...
```

## Tamanho Recomendado

- Altura: aproximadamente 50px (será ajustado automaticamente pelo CSS)
- Formato: PNG com fundo transparente (recomendado)
- Proporção: mantida automaticamente

## Paleta de Cores Aplicada

A paleta de cores do logo foi aplicada em todo o e-commerce:

- **Roxo Escuro** (#4B0082): Cor principal, botões primários, links
- **Roxo Claro** (#9370DB): Hover de botões primários
- **Laranja-Amarelo** (#FFA500): Destaques, badges, tags
- **Teal** (#20B2AA): Cor secundária, bordas do header, botões de checkout
- **Verde** (#32CD32): Sucesso, estoque disponível
- **Vermelho** (#FF0000): Erros, ações destrutivas
- **Azul Brilhante** (#1E90FF): Informações
- **Amarelo** (#FFFF00): Avisos, banners de ofertas

## Verificação

Após adicionar o arquivo LOGO.png, verifique se:
1. O logo aparece no header de todas as páginas
2. O logo é clicável e redireciona para a página inicial
3. O logo mantém suas proporções em diferentes tamanhos de tela
